# python封装exe可执行文件

## 工具 :

- ​	Pyinstaller

## 安装 :

**cmd下键入命令**

```python
pip install pyinstaller
```

## 打包封装 :

1. 进入需要封装的py文件的目录下

2. 按住`shift`键的同时鼠标右键选择在命令窗口打开

3. 输入封装命令

   ```python
   pyinstaller -F 文件名.py
   ```

4. 常用参数说明 :

   | 参数 | 描述                                                       |
   | ---- | ---------------------------------------------------------- |
   | -h   | 查看该模块的帮助信息                                       |
   | -F   | 制作独立可执行程序                                         |
   | -D   | 制作出的档案存放在同一个文件夹下（默认值）                 |
   | -w   | 指定程序运行时不显示命令行窗口（仅对 Windows 有效）        |
   | -c   | 指定使用命令行窗口运行程序（仅对 Windows 有效.且是默认值） |
   | -i   | 指定程序图标 后面跟ico文件路径                             |
   | -n   | 指定程序名称                                               |
